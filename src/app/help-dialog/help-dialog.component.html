Browsing Tiles

The default mode of the browser is browsing Sentinel 2 tiles. These are complete satellite images taken from the Sentinel program by ESA.
Tiles can be filtered using the following 4 inputs:

  • Maximum Cloud Cover: This is filtered by using the slider
  • Minimum File Size: This is chosen by using the text input box (note: most complete files are no larger than 800MB)
  • Date Range: which is selected by using the date widget
  • Bounding Box: When you enter search the only tiles that are returned will be ones that intersects with the bounding box of the map visible on your screen.

Once tiles have been searched for and selected they can be downloaded directly from the CEDA archive at http://data.ceda.ac.uk.

Patches

Tiles are broken down into thousands of smaller patches for land cover detection using machine learning.
These patches can also be searched for, or if a tile is selected you can view the patches that make it up by selecting the "View Patches" button.

Browsing Patches

By clicking the options button in the top left of the browser, Patch Search can be enabled.
This will let you search for Sentinel 2 patches instead of tiles.
Each patch is a section of a tile that has been given multilabels using a convolutional neural network.
Patches can be filtered using the following 3 inputs:

  • Multilabels: This is filtered by using the "Labels" drop down menu. You can choose as many labels as you want and only patches that are labelled with all selected labels are chosen.
  • Date Range: which is selected by using the date widget
  • Bounding Box: When you enter search the only patches that are returned will be ones that intersects with the bounding box of the map visible on your screen.

Once patches have been returned they can be selected and the tile that they come from can be downloaded directly from the CEDA archive at http://data.ceda.ac.uk.
You can also view the tile that any given patch came from by selecting it and clicking the "View Tile" button.

To learn more about Sentinel 2 vist: https://www.esa.int/Applications/Observing_the_Earth/Copernicus/Sentinel-2
